# Objetivo General
Crear una aplicación web, para satisfacer los requerimientos de la empresa Musync.sas, la cual permite a los usuarios: crear; actualizar; buscar y borrar listas de reproducción de música.
# Objetivo específicos 
- Crear un mockups de la aplicación web con las vistas de registro,log in, olvide contraseña, pantalla principal, y pantalla de administración crud. 
- Crear un frontend funcional para la aplicación web.
- Crear un backend que permita realizar y gestionar el almacenamiento de la información a ser desplegada en la aplicación web por medio de una api
- Configurar un servidor web para el lanzamiento de la aplicación web
